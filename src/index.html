Se utiliza el comando tsc --init para la obtener el tsconfig.json

Dentro de esta configuración ponemos en false lo siguiente:

  • sourceMap
  • declaration
  • declarationMap

Para que no aparezca el object cambiamos lo siguiente:

  • module: esnext
  • target: es2016

Maneras de inicializar un proyecto usando TS, utilizar los siguientes comandos:

  • npm init -y
  • npm install --save-dev typescript
  • npx tsc --init
  • En los archivos crear un archivo .gitignore
  • Crear un folder src y poner dentro los archivos, el index.html dejarlo fuera
